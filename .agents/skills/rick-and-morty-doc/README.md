# rick-and-morty-doc

A skill providing documentation and guidance for working with the Rick and Morty API.

## Overview

This skill contains information about:
- Rick and Morty API endpoints
- Response schemas
- Filtering and pagination
- Implementation patterns using ofetch

## Files

- `SKILL.md` - Quick reference for the skill
- `AGENTS.md` - Detailed API documentation for agents

## Usage

When implementing features that interact with the Rick and Morty API, load this skill:

```
skill(name: "rick-and-morty-doc")
```

This will provide relevant context about the API endpoints and response formats.
